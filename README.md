# gulimall

秋有鱼谷粒商城-基础篇

声明：需要前端代码的直接私信

#### 软件及版本

- IDEA 2020.3.2
- nacos-1.1.4
- node.js 10.16.3
- Mysql 8+
- VSCode
- 


#### 技术说明

1.  前端：Vue + ElementUi
2.  后端：SpringCloudAlibaba + MybatisPlus + Mysql + 阿里云OSS


#### 使用说明

1.  前端代码用VSCode打开，npm run dev 运行
2.  后端clone链接，IDEA修改个别数据调通运行
